# Simple Side Panel Extension

This is a simple browser extension that adds a side panel to the browser.

## Files

- `manifest.json`: The extension manifest file.
- `panel.html`: The HTML file for the side panel.

## How to Load the Extension

1. Open your browser and go to the extensions page (e.g., `chrome://extensions` for Chrome).
2. Enable "Developer mode".
3. Click "Load unpacked" and select the `extension` folder.
4. The extension will appear in your browser with a side panel.
